# 🚀 SpectraSearch - ProfilePulse

> **A modern profile search application that dynamically displays profile images and user information through real-time search interactions.**

---

## 🌟 Overview

**SpectraSearch - ProfilePulse** is an interactive profile search application built using HTML, CSS, and JavaScript.

The application allows users to search and instantly view profile images and details through a smooth and responsive interface. As users enter a name, the application dynamically updates the displayed profile information in real time, creating an engaging and seamless user experience.

This project was developed to strengthen core front-end development skills, including DOM manipulation, event handling, responsive design, and dynamic content rendering without relying on external frameworks.
